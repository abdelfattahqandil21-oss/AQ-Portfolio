# Debugging

1. Reproduce with smallest input before touching code
2. State hypothesis before making changes
3. Change one thing at a time, verify each
4. Check types, nullability, async boundaries first
5. Read full error messages + stack trace
6. Add structured logs before guessing
7. Fix root cause, not symptom
8. Clean up all debug logs before finishing
